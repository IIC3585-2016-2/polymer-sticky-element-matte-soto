# Sticky element component

Polymer component that emulates the `position: sticky;` CSS property.

## How to use

```html
<link rel="import" href="path/to/relative-sticky.html">

<relative-sticky>
  <div>
    <h1>This is the title</h1>
    <p> This is a subtitle </p>
  </div>
</relative-sticky>
```
